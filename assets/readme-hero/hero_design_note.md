# README Hero Design Note

- Project summary: PhysicsGuard is a Python core plus Codex skill for low-fidelity residual audits of physical simulation results.
- Target users: AI agents and engineers debugging large simulation models with exported or mapped signals.
- Core problem: complex models can fail through unit errors, sign mistakes, bad maps, broken balances, or wrong signal mappings.
- Core workflow: external simulation results -> mapped observed values -> PhysicsGuard residual audit -> suspicious blocks -> next signals/refinements.
- Hero tagline: AI-guided low-fidelity audits for physical simulation results.
- Visual concept: a left-to-right engineering audit pipeline with cards for external simulation, observed signals, PhysicsGuard audit, and next debug step.
- Image keywords: simulation audit, residual diagnostics, hierarchy, mapped signals, AI debugging, light technical product visual.

- README insertion position: after the H1 title.
